# **App Name**: 4arth Landing Page

## Core Features:

- Image Rendering: Faithfully render the provided image using HTML and CSS.
- Text Reproduction: Accurately reproduce all text from the image, including contact information and social media handle.
- QR Code Generation: Generate a functional QR code that links to the company's Instagram.
- Responsive Design: Ensure the page is responsive and displays correctly on different screen sizes.

## Style Guidelines:

- Background: Very light gray (#F0F0F0) to simulate paper and ensure high contrast.
- Text and Border: Black (#000000) for clear readability and accurate reproduction of the image.
- Font: 'Literata' serif. A literary feel.
- Use a centered layout with a white container to mimic the image's appearance.
- Include simple icons for phone and address details.
